¿Qué diferencias y semejanzas hay entre Java y JavaScript?

Diferencias
  • En Java se necesita un kit de desarrollo y un compilador mientras que Javascript sus programas
    se interpretan con el navegador web.
  • Con Java se pueden hacer varios programas variados mientras que con Javascript solo se pueden
    hacer programas para que se ejecuten en páginas web
  • Javascript esta basado en un estándar abierto sin dueño y por ende cualquiera puede implementarlo
    en su sistema, mientras que Java le pertenece a una empresa, por lo que su enfoque es comercial y
    paticular.
  • Java es un lenguaje que esta orientada a objetos y Javascript no, por lo que no se necesita conocimiento
    del tema, aunque si lo permite.

    • Semejanzas
  • La colaboración de los cradores de ambos lenguajes.
  • Sintáxis idéntica.

    • ¿Qué métodos tiene el objeto Date? (Menciona al menos 5*)

  • getDate(): Devuelve el número del día del mes.
  • getMonth(): Devuelve el mes.
  • getSeconds(): Devuelve los segundos.
  • setMinutes(): Cambia los minutos.
  • toTimeString(): Convierte la parte de tiempo de un objeto date a una cadena.

    • ¿Qué métodos tienen los arreglos? (Menciona al menos 5*)

  • concat(): Une dos o más arrays.
  • join(): Junta los elementos de un array en una cadena con un separador.
  • reverse(): Invierte el orden de los elementos de un array.
  • sort(): Ordena los elementos de un array, como strings.
  • split(): Convierte un string en un array.

    • ¿Cómo se declara una variable con alcance local dentro de una función?

    Se declaran con var (var x = "Hola mundo";).

    ¿Qué implicaciones tiene utilizar variables globales dentro de funciones?

    Son accesibles desde cualquier parte del programa y por edne para todas las funciones, aunque esten dentro
    de éstas.
