¿Por qué es una buena práctica usar JavaScript para checar que sean válidos los inputs de las formas antes de enviar los datos al servidor?

Para que se envíen los datos de manera correcta al servidor. De esta manera el programa funcione correctamente y los datos del usuarios quedan protegidos.

¿Cómo puedes saltarte la seguridad de validaciones hechas con JavaScript?

Deshabilitando javascript del navegador, hacer el form.submit u otra cosa colocando código javascript en la url.

Si te puedes saltar la seguridad de las validaciones de JavaScript, entonces ¿por qué la primera pregunta dice que es una buena práctica?

Ya que es más amigable para el usuario, porque le permite tener un experiencia amigable cuando se le validan sus datos o cuando se equivoca, además de que los datos se guardan con fácilidad en el sistema y nadie accede a ellos.
